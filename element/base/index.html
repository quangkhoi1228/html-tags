# base

Định nghĩa URL cơ bản cho tất cả đường dẫn tương đối của một trang web. Nên được đặt trong thẻ <head>.

Ví dụ: Copy

Thẻ base trong HTML
{% highlight html %} Thẻ base trong HTML {% endhighlight %}

href

Định nghĩa đích cơ bản của tất cả liên kết của trang web.

"{{site.url}}"

Bạn có thể gán một đường dẫn tuyệt đối.

target

Định nghĩa tab hay cửa sổ nào sẽ được mở khi click liên kết.

"_blank"

Mở 1 bối cảnh duyệt web mới của trình duyệt thường là 1 tab mới.

"_self"

Mở tại tab hiện tại.

"_parent"

Mở tại bối cảnh duyệt web cha của phần tử đang duyệt, hoặc _self nếu không có bối cảnh duyệt web cha.

"_top"

Mở tại bối cảnh duyệt web đầu, hoặc _self nếu không có bối cảnh đầu.
