# iframe

Định nghĩa một container for một nested bối cảnh duyệt web: you can include một trang web within another trang web.

Ví dụ: Copy

{% highlight html %}{% endhighlight %}

src

Định nghĩa URL of the nested trang web.

"https://cssreference.io"

Bạn có thể type any URL. Be aware that some websites sẽ not allow to be iframed.

allowfullscreen

Cho phép the browser to show the iframe in fullscreen with JavaScript.

Không yêu cầu giá trị.

height

Định nghĩa height of the iframe.

"50"

The height in pixels.

width

Định nghĩa width of the iframe.

"150"

The width in pixels.
