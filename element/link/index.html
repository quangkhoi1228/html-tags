# link

Định nghĩa một link between the trang web hiện tại and một external link or resource.

Ví dụ: Copy

{% highlight html %}{% endhighlight %}

href

Định nghĩa URL of the link.

"{{site.url}}/css/website.css"

Bạn có thể pass một đường dẫn tuyệt đối.

"/css/website.css"

Bạn có thể pass một đường dẫn tương đối liên quan với tên miền gốc.

rel

Định nghĩa một link type, explaining how the link relates to the trang web hiện tại.

"stylesheet"

The link is một stylesheet.

"icon"

The link is một favicon.

"author"

The link is the web page's author website.

"next"

The link is the next page.

type

Định nghĩa type of the linked resource.

"text/css"

The link is một CSS file.

"text/html"

The link is một văn bản HTML.
