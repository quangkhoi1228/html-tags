value

Định nghĩa value of the progress bar, on the scale defined by the max attribute.

Bắt buộc.

"0.7"

Bạn có thể sử dụngn n decimal and negative numbers. It must be between the min and max values.

"-42"

If you nguồn n n một value that is lower than the minimum, the progress bar sẽ be empty.

"63"

If you nguồn n n một value that is higher than the maximum, the progress bar sẽ be full.

max

Định nghĩa maximum value possible on the progress bar.

"1"

Default.

Bạn có thể sử dụngn n decimal and negative numbers.

"100"
