# source

Định nghĩa một source for một <audio>, <video>, or <picture> element.

Ví dụ: Copy

{% highlight html %} {% endhighlight %}

src

The URL where the media is hosted.

"/images/tiramisu.jpg"

Bạn có thể sử dụngmột relative URL.

"{{site.url}}/images/traffic.jpg"

Bạn có thể sử dụngmột đường dẫn tuyệt đối.

srcset

Định nghĩa một list of different sources for the same media. The browser sẽ choose the best one to use.

"/images/sunset-360.jpg 360w,
/images/sunset-720.jpg 720w,
/images/sunset-1440.jpg 1440w"

Bạn có thể sử dụngmột width descriptor like 360w. This value is divided by one of the nguồn sizes (defined in the sizes attribute) to obtain the pixel density.

@html

sizes

Định nghĩa một list of different nguồn sizes. Bạn có thể combine each of them with một media query.

"(min-width: 800px) 1440px, 720px"

The browser sẽ nguồn the 1440px image (defined in srcset) if the viewport is larger than 800px.
It sẽ nguồn the 720px otherwise.

@html

type

Định nghĩa MIME type of the source.

"image/jpg"

Bạn có thể only nguồn audio, video, or picture MIME types.

media

Định nghĩa một media query for một <picture> source.

"(min-width: 800px)"

The media sẽ only be used on viewports larger than 800px.
